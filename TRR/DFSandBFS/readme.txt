DFSandBFS description:
Bài A: 
- File DFSandBFSA: in ra số thành phần liên thông (bài này chỉ cần chạy).
- File Direct: ý 2 của bài A, in ra đường đi, thầy nhập 2 chữ là được ạ. 
(Hoặc không e có C in C out).
Bài B: 
- File DFSandBFSB: e làm cả 3 ý cùng trong này
- Thầy có thể nhập input hoặc e có file in out ạ
- a: số tp liên thông mạnh
- b: các đỉnh (chữ) cùng tp liên thông mạnh vs chữ nhập từ input (bao gồm cả nó).
(nếu thầy ko in đỉnh nhập thì e có code ở dưới)
- c: đường đi (như bài a ở trên).
